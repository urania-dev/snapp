Retrieve a list of Snapps

This endpoint returns a list of Snapps owned by the authenticated user. The response includes basic information about each Snapp and optionally the total count of Snapps.

Authorizations:
bearerAuth
query Parameters
page
integer
Default: 1

Page number

limit
integer
Default: 10

Number of items per page

query
string

Search query for short codes or original URLs

sortBy
string

Field to sort by (created_at, expires_at, original_url, short_code, usages)

dir
string
Enum: "asc" "desc"

Sorting direction (asc or desc)

count
boolean

Include the total count of Snapps

Responses

Rest Api Interface

https://snapp.li/api/snapps

Response samples

  • 200
  • 401
  • 500
Content type
application/json

Retrieve a specific Snapp

This endpoint retrieves information about a specific Snapp identified by its ID or short code. The response includes basic information about the Snapp and optionally its usage details. Pagination is related to Url Usages if included in the query

Authorizations:
bearerAuth
path Parameters
id
required
string
Default: "1023938d-0294-4d35-b5c0-08339bf1738c"

Snapp ID (UUID)

query Parameters
usages
boolean

Include the UrlUsages to Snapp

page
integer
Default: 1

Page number

limit
integer
Default: 10

Number of items per page

sortBy
string

Field to sort by for URLs (timestamp, country, city, region, user_agent, referrer, language, browser, os, device, cpu, id)

dir
string
Enum: "asc" "desc"

Sorting direction (asc or desc)

Responses
